# 📌 Overview
CareerPilot AI is an AI-powered Career Intelligence platform designed to help students and job seekers analyze job market trends, salary insights, skill demand, and personalized career growth opportunities.
The platform combines:

* Data Science
* Machine Learning
* NLP
* Forecasting
* Analytics dashboards
* Interactive visualizations
into a single modern SaaS-style web platform.

# 🚀 Installation
## Clone Repository

```bash
git clone https://github.com/your-username/CareerpilotAI.git
```


## Frontend Setup

```bash
cd frontend
npm install
npm run dev
```


## Backend Setup

```bash
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload
```
